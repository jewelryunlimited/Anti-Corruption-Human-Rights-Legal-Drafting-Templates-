# Legal Drafting Templates for Anti-Corruption & Human Rights

## Purpose
This repository provides open-source legal drafting templates for NGOs, civic defenders, and victims of corruption. It includes petitions, affidavits, and complaints in multiple languages (English, Hindi, Indonesian, Arabic).

## Usage
- NGOs and activists can adapt these templates for case filings.
- Victims can use them to prepare complaints and petitions.
- Students can learn standardized drafting methods.
